# PolyhydraGames.OpenLibrary

OpenLibrary API integration for book data, covers, and search.

## Release

- GitHub Packages is the default package host.
- Use `./pub.sh` to restore, test, pack, and publish.
- Publish to nuget.org only when you explicitly want a public release.

## Contents

- `PolyhydraGames.OpenLibrary` - main client
- `PolyhydraGames.OpenLibrary.Data.Dapper` - Dapper persistence helpers
- `PolyhydraGames.OpenLibrary.Test` - tests
